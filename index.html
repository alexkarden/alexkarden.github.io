TG бот - уведомления о снижении цен на товары

Скрипт, периодически парсит сайты и записывает актуальные цены в базу PostgreSQL. Сравнивает цены и рыссылает уведомления пользователям в случае изменения цены. Есть возможность задавать порог цены, объединять товары в пулы. Импорт истории цен в файл exel. Графическое представление истории цен. Подключен Redis для ускорения работы бота. Скрипт выполняется в контейнере Docker на сервере Linux, с удаленным управлением через SSH.

Python 3.12, asyncio, aiogram, APScheduler, requests, beautifulsoup4, matplotlib, pandas, asyncpg, redis, Docker
